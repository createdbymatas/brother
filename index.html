Spausdintuvo instrukcija

Brother DCP-L2620DW

Pasirink:

Skenavimas

Prieš pradedant skenavimą patvirtink, kad:

Spausdintuvas yra įjungtas
Wi-Fi lemputė spausdintuvo skydelyje šviečia žalia spalva (nemirksi)
Į skenerio vietą įdėtas lapas, kuris turi būti nuskenuotas

Pasirink:

Skenuotą dokumentą išsaugoti...

kompiuteryje
telefone

1. Paspausk Scan mygtuką spausdintuvo skydelyje.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Scan: Select ▲▼

(Skenavimas: Pasirinkti ▲▼)

2. Paspausk bet kurį iš šių mygtukų: ▲▼.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Select ▲▼ or OK

to PC

(Pasirinkti ▲▼ ar OK / į kompiuterį)

3. Paspausk OK mygtuką.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Select ▲▼ or OK

File

(Pasirinkti ▲▼ ar OK / Failas)

4. Paspausk OK mygtuką.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Scan to File

Press Start or Options

(Skenuoti failą / Paspausti Start (Paleisti) ar Options (Parinktys))

5. Paspausk ▬▬ Start mygtuką.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Connecting to PC

(Jungiamasi prie kompiuterio)

6. Palauk, kol spausdintuvas prisijungs prie kompiuterio.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Scanning

(Skenuojama)

7. Palauk, kol spausdintuvas nuskenuos lapą.

Spustelėk Toliau , kai spausdintuvo ekranas rodo:

Next Page?

▲ Yes ▼ No (Send)

(Sekantis puslapis / ▲ Taip ▼ Ne (Siųsti))

7.1. Jei kartu į tą patį failą nori išsaugoti dar vieną (kelis) lapą (-us), paspausk ▲ mygtuką;
jei ne - praleisk šį žingsnį.

Spustelėk Toliau , kai spausdintuvo ekranas vėl rodo:

Next Page?

▲ Yes ▼ No (Send)

(Sekantis puslapis / ▲ Taip ▼ Ne (Siųsti))

PASTABA: Spausdamas ▲ mygtuką gali pridėti tiek lapų, kiek papildomai reikia nuskenuoti ir pridėti į tą patį failą.

8. Spausdintuvo ekrane uždaryk skenavimo meniu naudojant ▬▬ Stop/Exit mygtuką.

9. Spausdinimas baigtas.

Kompiuterio ekrano apačioje turi pasirodyti pranešimas Completed (Baigta).
Paspaudus mygtuką Open Folder (Atidaryti aplanką) bus atidarytas kompiuterio aplankas Dokumentai, kuriame bus sukurtas failas.
Dokumento pavadinimas yra ScanMMMM-mm-dd_vvmmss.pdf (pvz.: Scan2025-07-28_140130).

Jei spausdintuvo ekranas rodo:

Scan to File

No PC found.

(Skenuoti failą / Kompiuteris nerastas.)

Uždaryk skenavimo meniu spausdintuvo ekrane naudojant ▬▬ Stop/Exit mygtuką,
kompiuteryje atidaryk programą Brother iPrint&Scan,
atlik visus veiksmus iš naujo.
